# web-clip-helper v0.4.0 反馈

**日期:** 2026-05-12
**环境:** WSL2 / Ubuntu / Python 3.11.15
**当前版本:** v0.3.2 (回退后正常工作)

---

## 问题1: v0.4.0 Linux 二进制无法启动

**现象:**
- SHA256 校验通过 (`a3640249ea93b0a6f032f11cf29cdbc51b94f63363b7b3e1c490b0ff6f250ffa`)
- `ldd` 依赖全部满足
- 运行时报错:

```
Traceback (most recent call last):
  File "pyimod02_importers.py", line 649, in install
  File "pyimod01_archive.py", line 65, in __init__
pyimod01_archive.ArchiveReadError: PYZ magic pattern mismatch!

RuntimeError: Failed to setup PYZ archive reader!
```

**分析:**
- PyInstaller PYZ archive magic pattern 不匹配
- 可能是构建环境的 PyInstaller 版本与 WSL2 运行时兼容性问题
- v0.3.2 二进制 (28MB, 同一环境构建?) 运行正常
- v0.4.0 二进制 (23.9MB) 显著小于 v0.3.2 (28MB)，可能使用了不同 PyInstaller 版本或打包选项

**建议:**
- 检查 PyInstaller 构建环境是否变更
- 考虑在 CI 中增加 smoke test (在干净 Ubuntu 上运行 `agent info`)
- 同时提供 pip 包, 避免 PyInstaller 兼容性问题

---

`agent update check` 返回 "Package not found on PyPI"。如果能发布到 PyPI，用户可以通过 `pip install` 安装，绕过 PyInstaller 二进制兼容性问题。
